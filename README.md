# QuestLog - Adventure Assistant

QuestLog is a robust command-line application designed to help players manage their adventure quests and inventory. It features dual-mode execution, batch file processing, and smart inventory analysis.

## 🚀 Features

- **Interactive Mode**: A continuous command-line session for real-time management.
- **Quest Planning**: Automatically identifies which quests are completable with your current items.
- **Gap Analysis**: Tells you exactly what items and quantities you are missing for a specific quest.
- **Batch Processing**: Process multiple inventory updates at once using text files.
- **Persistence**: Automatically saves and loads data from JSON files.
